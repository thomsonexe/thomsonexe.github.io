  • Find the hidden flag inside

hash_hunt

A credential database was dumped during a breach. The passwords are hashed — but not all algorithms are equal. Identify the type and crack the target entry to recover the plaintext.

Objectives

  • Identify the hash algorithm used in the database
  • Crack the target hash to recover the plaintext password
  • Find which other account reuses the same password

Get the Challenge Files

$ cd ~/challenges/hash-hunt/
$ cat README.txt

Tools You'll Need

  • hashid or hash-identifier
  • john --format=raw-md5 --wordlist=wordlist.txt passwd.db
  • grep for cross-referencing entries
Open Terminal
